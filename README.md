# Discord-Convocations

## Fichier convocation.py

### Téléchargement

Vous pouvez copier le contenu du code ou bien télécharger le fichier .py directement

### Modifications à apporter

Pour faire fonctionner le fichier avec votre bot et ajouter la commande /convoquer, il vous faudra modifier les lignes
- 6
- 7
- 22 (si nécessaire uniqument)

Afin de remplacer les variables et les permissions nécessaires pour déclancher la commande.

## Interactions avec Discord

Grâce à ce fichier et à l'exécution de la commande, vous pourrez "convoquer" des membres de votre serveur et leur demander à ouvrir un ticket.
Une fis la commande exécutée n'importe où sur votre serveur, un message sera envoyé dans le salon public que vous avez paramétré avec en contenu
le membre mentionné, la raison de la convocation ainsi que le salon ticket que vous avez paramétré aussi dans le fichier.
Un message privé sera aussi envoyé au(x) membre(s) concernés, sauf si vous décidez de supprimer cette fonctionnalité.
